
1. cookie模拟注册和登录? (单用户)
a. 注册所有信息全都正确  => 将对应的数据通过后端存储到数据库, 跳转到登录 (数据库存储数据)
b. 登录时进行验证  用户输入的用户名和密码  与  数据库中存储的用户名和密码对比
   成功 => 记录当前登录的用户,跳转主页面
   失败 => 用户名或密码有误
c. 主页面 表示用户身份
    用户登录  => 欢迎,xxx
    没有用户登录  => 亲，请登录 免费注册
